Zotero
Zotero is a free tool to help you collect, organize and share research.
A Unified Zotero Experience
Zotero has been consoldiated into a single software that is compatible with all browsers via the Zotero Connector plugin.
Zotero Connector
Zotero Connector is a browser plugin that integrates Zotero into your web browser and helps you save content to your Zotero library.

Zotero is available for various Linux distributions. The following steps will show you how to download and configure Zotero.
Installing Zotero on Chromebook
Zotero isn't available for Chrome OS but can be used in its Linux version on a Chromebook running Linux.
Check for updates
You can check for and download updates for Zotero within the software itself.
Setting up an online account
Zotero can be used without an account but creating an online account will allow you to sync and access your library from anywhere. You will aslo be able to join groups and backup your attached files. Creating a Zotero account is free.
Data Syncing
Data Syncing merges the Zotero content on your local computer with the content on the Zotero servers.
File Syncing
File attachements aren't synced with data syncing.
Setting up file syncing with Zotero Storage
File syncing allows you to access file attachments online and from any computer. Zotero Storage provides each user with 300Mb of free storage and is the recommended option for file syncing.
Setting up file syncing with WebDAV
File syncing allows you to access file attachments online and from any computer. WebDAV is a standard protocoal for transferring files on the web and can be used to sync your Zotero files.
Linked Files
Linked files are shortcuts or links that point a program to the location of the actual file.
Copying the Zotero data directory
You can back up your Zotero data without using Zotero servers by manually copying your Zotero files to a location of your choice.

Quick Search
Quick searches provide a fast way to find items in a library or collection.
Advanced Search
Advanced searches offer more and finer control than quick searches, and allow you to make saved searches.
PDF Full-Text Indexing
Full-text PDF indexing allows embedded text within PDFs to be searched with Quick Search (via the “Everything” option) and Advanced Search (via “Attachment Content”).
Sorting Items
Items in the center pane can be sorted by various properties, such as their title, creators, or the date they were added to your library.
Citation Styles
Zotero ships with several popular citation styles for creating citations and bibliographies, and over 8,100 additional styles can be found in the Zotero Style Repository: All these styles are written in the Citation Style Language (CSL), , a format also supported by Mendeley, Papers, and many other programs.

Reports
Reports are simple HTML pages that give an overview of the item metadata, notes, and attachments of the selected items. You can print them, post them to the web, and email them.
Word Processor Plugins
Word Processor Plugins are the easiest way to automatically generate bibliographies from Zotero. Generate dynamic bibliographies, intext citations, and footnotes with Google Docs, Word, and LibreOffice.
